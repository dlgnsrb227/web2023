Vue 개념

React는 Facebook에서 개발한 오픈소스 자바스크립트 라이브러리로, 사용자 인터페이스를 만들기 위해 사용됩니다.

React는 기존의 HTML, CSS 및 JavaScript 기술과 함께 사용됩니다. React는 빠르고 효율적인 UI 개발을 가능하게 하며, 동적인 웹 애플리케이션을 쉽게 작성할 수 있도록 해줍니다.

React는 컴포넌트 기반 아키텍처를 사용하며, 재사용 가능한 UI 요소를 만들어낼 수 있습니다. 또한, React는 Virtual DOM이라는 기술을 사용하여 UI 업데이트를 빠르고 효율적으로 처리할 수 있습니다.

Vue 설치

Node.js 설치
https://nodejs.org/ko

먼저 Node.js를 설치해야 합니다. Vue.js는 Node.js에 기반하므로, Node.js가 먼저 설치되어 있어야 합니다. 다음 사이트에서 Node.js를 다운로드하여 설치할 수 있습니다

npm install -g @vue/cli

명령 프롬프트에서 다음 명령을 실행하여 Vue CLI(Command Line Interface)를 전역으로 설치합니다. Vue CLI는 Vue.js 프로젝트를 관리하는 데 사용되는 도구입니다.

Vue CLI v5.0.8
    ? Please pick a preset: (Use arrow keys)
    > Default ([Vue 3] babel, eslint)
    Default ([Vue 2] babel, eslint)
    Manually select features

Vue CLI 설치가 완료되면, 새로운 Vue 프로젝트를 생성할 디렉토리로 이동합니다. 예를 들어, C:\projects 디렉토리에 새로운 Vue 프로젝트를 생성하고 싶다면 다음 명령을 실행합니다.

cd C:\projects

이제 다음 명령을 실행하여 새로운 Vue 프로젝트를 생성합니다. 프로젝트 이름은 원하는 대로 지정할 수 있습니다.

vue create my-vue-app
