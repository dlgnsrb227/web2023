01. 데이터 이펙트 : 1부터 100까지 출력하기

for문을 이용해서 1부터 100까지 출력하는 방법입니다.

 for(let i=1; i<=100; i++){
    console.log(i);
}
결과 확인하기
1, 2, 3, 4, 5 ~ 99, 100

02. 데이터 이펙트 : 0부터 50까지 출력하기

범위 값은 초기값과 조건식으로 변경 가능하다.

 {
    let x = 100;
    let y = 200;
    let z = "javascript";

    x = 300;
    y = 200;
    z = "react";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
